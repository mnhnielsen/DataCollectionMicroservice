Data Providing Service!

This is the service where teams can get data collected from the system overall.

Some of the endpoints the user can access:

GET /users --> Get a list of users

GET /users/(id) --> Get a specific user's profile

GET /users/{id}/songs --> Get a specific users song listening history

The rest of the endpoints can be viewed at the teams gitlab repo:

https://gitlab.sdu.dk/semester-project-e2021/team-05/template/-/wikis/Appendix#api-endpoints
